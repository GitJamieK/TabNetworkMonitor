# Privacy Policy for Tab Network Monitor

Tab Network Monitor is a browser extension that displays estimated network activity for open browser tabs.

## Data the extension uses

Tab Network Monitor may access the following browser data:

- Tab titles
- Tab URLs
- Network request activity
- Estimated network usage per tab

This information is used only to display network activity inside the extension interface.

## Local processing

All data is processed locally in the user’s browser. Tab Network Monitor does not send browsing data, tab data, URLs, or network activity to any external server.

## Data sharing

Tab Network Monitor does not sell, share, transfer, or disclose user data to third parties.

## Data storage

Tab Network Monitor does not store personal browsing data on external servers. Any temporary data used by the extension is handled locally in the browser for the purpose of displaying network usage.

## Remote code

Tab Network Monitor does not use remote code. All extension code is included in the extension package.

## Contact

For questions about this privacy policy, contact the developer through the extension’s Chrome Web Store listing or GitHub repository.
